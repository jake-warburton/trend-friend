# Repurposing Chain

1 topic -> 5 platform-native outputs. Not reformatted copies — each thinks differently.

## The Chain

### Step 1: Extract the Story

Given a topic (or Signal Eye data export), identify:
- The surprising fact (what moved, by how much)
- The "so what" (why should the audience care)
- The evidence (which sources corroborate, score breakdown)

### Step 2: Write X Post First

X forces brevity. Write the tightest version first.
- Apply [[hooks]] — pick the hook type that fits the story
- Apply [[platform-tone]] X rules
- Max 280 chars for single tweet, or plan a 3-5 tweet thread
- This becomes the "seed" for all other platforms

### Step 3: Expand to LinkedIn

Take the X seed and expand:
- Add personal context: "I was looking at this week's data and..."
- Add the fuller explanation — why this matters for business/product decisions
- Add 2-3 bullet points of supporting data
- End with a question for engagement
- Target: [[audience/founders-pms]] and [[audience/seo-teams]]

### Step 4: Build Instagram Carousel

Take the core insight and make it visual:
- Slide 1: bold claim from the X hook (big text, Signal Eye branding)
- Slides 2-4: supporting data points, one per slide, with score visuals
- Slide 5: the "so what" — what to do with this information
- Slide 6: CTA — "follow for weekly trend data" or "link in bio"
- Caption: conversational version of the LinkedIn narrative (shorter)

### Step 5: Script TikTok

Take the surprising fact and make it a 30-45 second screen recording:
- Open with: "okay so [hook from step 2]"
- Show the actual Signal Eye dashboard while narrating
- Point at the specific trend, show the score, show the sources
- End with: "follow for more trend intelligence"
- No editing needed. Raw > polished

### Step 6: Draft Blog Post (when active)

Take the LinkedIn version and expand into SEO-targeted long-form:
- Research target keyword using the topic + "trends" / "emerging" / "2026"
- Structure: hook -> context -> Signal Eye data -> analysis -> implications -> CTA
- Embed dashboard screenshots
- Link to related Signal Eye pages
- Target: [[audience/seo-teams]] who will share internally

## Content Ratios

Not every topic needs all 5 platforms. Use this guide:

| Signal strength | Platforms |
|-----------------|-----------|
| Breakout trend (big mover) | All 5 |
| Interesting data point | X + LinkedIn |
| Behind-the-build update | X + TikTok + IG stories |
| Weekly roundup | X thread + LinkedIn + IG carousel |
| Quick observation | X only |
