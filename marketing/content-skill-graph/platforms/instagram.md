# Instagram

Visual-first. Data stories as carousels.

## Format Rules

- **Carousels** (primary): 5-7 slides. bold claim on slide 1, data on middle slides, CTA on last
- **Reels**: 30-60s. same as TikTok content, can cross-post
- **Stories**: behind-the-scenes, quick polls, dashboard screenshots
- Caption: 150-300 words. conversational. include CTA

## Slide Design Principles

- Slide 1: one bold statement in large text. Signal Eye branding subtle
- Data slides: one number per slide. big font. minimal decoration
- Use consistent colour scheme from the Signal Eye dashboard
- Screenshot the actual dashboard — it's visually distinctive
- Last slide: "follow for weekly trend intelligence" + website

## Style

- Caption reads like you're explaining to a friend
- Use emojis sparingly — 1-2 per caption max
- Apply [[brand-voice]] — keep it casual and data-backed

## What Works

- "Top 5 trends this week" carousels with scores
- Before/after: "this trend went from nowhere to #3"
- Source comparison: "what Reddit sees vs what GitHub sees"
- Dashboard screenshots with annotation

## Audience Here

Mix of [[creators]] and [[founders-pms]]. More visual learners.

## Repurposing

Build from X seed per [[repurpose]] chain. Core insight becomes slide 1. Supporting data becomes slides 2-5.
